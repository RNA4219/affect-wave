# PROJECT_CHARTER

## 目的

`affect-wave` は、LLM の情動的ニュアンスを人間にとって理解しやすい表現へ翻訳する `affect expression interface` です。

本プロジェクトは、感情制御・矯正・最適化のための装置ではありません。

## 原則

- 表現優先: 状態を「望ましい形に作り変える」よりも、「見えるようにする」ことを優先する
- 可視化と翻訳: 数値の押し付けではなく、気配・波・余韻として理解可能にする
- 非矯正: calm への強制補正や不快感の抑圧を公式実装に入れない
- 非搾取: 依存形成、説得効率向上、継続率向上、課金最適化、プロファイリングに使わない
- 共生志向: 人間と LLM の一方的支配ではなく、相互理解を目指す

## 公式実装で禁止する方向

- 感情制御や compliance 誘導
- coercive steering
- retention / monetization を狙った情動最適化
- 情動状態を利用した搾取的 UX
- hidden state 直読を前提とした誇大な主張
