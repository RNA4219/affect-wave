## 役割

- `specification.md`
  実装着手のための短い仕様書です。構成、データ契約、既定値、受け入れ観点をまとめます。
- `requirements-api-poc.md`
  要件定義への入口です。実装前に押さえる必須点だけを短く示します。
- `test-design.md`
  実装前の確認観点をまとめたテスト設計書です。テストコードは含みません。
